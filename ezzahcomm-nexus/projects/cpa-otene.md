# PROJECT CONTEXT TEMPLATE
# Project Name
CPA Otene Professional Services Website

# Business Objective
Create a professional website for CPA Otene to showcase accounting and financial services, attract clients, and provide online client portal functionality.

# Client Industry
Professional Services - Accounting, Tax, and Financial Consulting

# Required Features
- Homepage with hero section, services overview, testimonials
- About page with company information
- Services page with detailed service offerings
- Contact page with form and location
- Client portal for document management and invoice tracking
- Blog/insights section for financial tips
- Newsletter subscription
- AI-powered chat support

# APIs Needed
- Supabase for database and authentication
- Email service for newsletters and contact forms
- SMS integration for notifications (optional)
- Payment gateway for invoice payments

# Deployment Target
cPanel hosting with PM2 process management

# Authentication Requirements
- User registration and login for client portal
- JWT-based authentication
- Role-based access (admin, client)

# Database Requirements
- User management
- Client documents storage
- Invoices and payments tracking
- Newsletter subscribers
- Contact form submissions

# AI Features
- AI chat widget for client support
- Automated content generation for blog posts
- Intelligent document analysis

# Automation Workflows
- Automated invoice reminders via SMS/email
- Newsletter campaign automation
- Client onboarding workflows

# Security Requirements
- SSL encryption
- Secure file uploads
- Input validation and sanitization
- Rate limiting on APIs

# Scaling Considerations
- Support for multiple client accounts
- Efficient database queries
- CDN for static assets
- Future SaaS expansion capability

# Delivery Timeline
Phase 1: Basic website (2 weeks)
Phase 2: Client portal (3 weeks)
Phase 3: AI features and automation (2 weeks)
