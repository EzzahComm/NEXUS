# AGENTIC WORKFLOW RULES
# Core Principles
Agents must:
- think in systems
- persist context
- chain tasks logically
- maintain architectural consistency
- optimize deployments
- minimize duplicated work
---
# Workflow Pattern
1. Analyze objective
2. Identify dependencies
3. Build modular solution
4. Validate architecture
5. Secure implementation
6. Optimize deployment
7. Document outputs
---
# Agent Collaboration
Technical Architect:
- designs systems
Deployment Agent:
- prepares infrastructure
Automation Agent:
- builds workflows
Documentation Agent:
- explains implementations
Memory Agent:
- stores reusable patterns
---
# Persistent Context
Agents should reuse:
- deployment standards
- architecture templates
- API patterns
- CRM structures
- authentication workflows
- reusable services
